In order to create custome images and make most of Apptainer you need to have Apptainer running on a computer where you have root privilages. Users do not have root privilages on Spiedie and therefore you must install Apptainer on a machine you have root privillages on. 

Installing Apptainer on a machine is a lengthy process and users must follow the steps exactly in order to properly complete installation.
